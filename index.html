Lingua

Introducing the Lingua programming language

Lingua was developed to bridge the gap between powerful, expressive languages such as C/C++ and readable, feature-rich languages as Python.

Lingua is fast, feature-full, and open source making writing beautiful software efficient and painless.

See it in action:

func factorial(num: int) -> int:
    @pre:
        ensure num >= 0
		
	fact: int = 1
	
	for x in [0...num]:
		fact *= x

    return fact

func main():
    result: int = factorial(5)

    print(result)
